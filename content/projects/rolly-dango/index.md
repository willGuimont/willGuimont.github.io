Rolly Dango is an isometric rolling puzzle made with [WASM-4](https://wasm4.org/) for the WASM-4 Game Jam.

Some key points of the project:

- We made our own entity-component system from scratch.
- To save cartridge space, we built our own Huffman coding algorithm.
- We built a custom level editor with [p5.js](https://p5js.org/).
- We made all the sprites with [Aseprite](https://www.aseprite.org/).
